Greylock SoC

Greylock is an SoC, it is comprised of the Titan X1 CPU and a number of peripherals.

Register Interface Description

Receive/Transmit Buffer Register (RBR/TBR) / Divisor Latch LSB Register (DLL)
Offset from base address: 0x0
Field (DLAB=0) Bits Default Value Description
RBR/TBR 7:0 N/A Read the received data byte, or write the data byte to transmit
Field (DLAB=1) Bits Default Value Description
DLL 7:0 Set for 115200 BAUD Divisor Latch LSB
Interrupt Enable Register (IER) / Divisor Latch MSB Register (DLM)
Offset from base address: 0x4
Field (DLAB=0) Bits Default Value Description
RDAI 0 0 Enable receive data available interrupt
TFEI 1 0 Enable transmit FIFO empty interrupt
Field (DLAB=1) Bits Default Value Description
DLM 7:0 Set for 115200 BAUD Divisor Latch MSB
Line Control Register (LCR)
Offset from base address: 0xC
Field Bits Default Value Description
DLAB 7 0 0 : RBR, THR, and IER are accessible. 1 : DLL and DLM are accessible.

Example code for setting the BAUD rate

Relevant definitions 
.uart_base_address_hi     h0 ; set this to the top 16-bits of the UART base address
.uart_base_address_lo     h0 ; set this to the bottom 16-bits of the UART base address
.uart_reg_offset_dll      h0 ; DLL register byte offset
.uart_reg_offset_dlm      h4 ; DLM register byte offset
.uart_reg_offset_lcr      hc ; LCR register byte offset
Sample code 
; put the UART base address into r1
mov     r1,     uart_base_address_lo
movh    r1,     uart_base_address_hi
; set the baud rate to 115200
ldr     r0,     [r1, uart_reg_offset_lcr] ; read the lcr
or      r0,     h8                        ; set DLAB bit to 1
str     r0,     [r1, uart_reg_offset_lcr] ; write it back
mov     r0,     uart_divisor_115200       ; set r0 to the correct divisor for 115200 baud
str     r0,     [r1, uart_reg_offset_dll] ; write the divisor's lsb
lsr     r0,     d8                        ; move the msb into the bottom byte
str     r0,     [r1, uart_reg_offset_dlm] ; write the divisor's msb
ldr     r0,     [r1, uart_reg_offset_lcr] ; read the lcr
and     r0,     h7                        ; set DLAB bit to 0
str     r0,     [r1, uart_reg_offset_lcr] ; write it back
